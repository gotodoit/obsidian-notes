# 🚀 GitHub Trending (2026-01-29)

## 1. [google/generative-ai-js](https://github.com/google/generative-ai-js)
> [!abstract] Google 官方推出的 JavaScript SDK，用于在 Web 应用或 Node.js 环境中轻松集成 Gemini 系列生成式 AI 模型。

### 💡 核心特性
- 支持文本生成、多模态输入（图片和文本混合）以及结构化输出。
- 内置多轮对话状态管理，并支持流式响应 (Streaming) 以实时展示模型输出。
- 提供函数调用 (Function Calling) 功能，使模型能够触发外部逻辑或查询实时数据。

### 🛠️ 技术栈
`TypeScript`, `JavaScript`, `Node.js`, `Browser Fetch API`

### ⭐ 亮点分析
该项目是接入 Google Gemini 最权威的 JS 开发工具包，屏蔽了底层 API 调用复杂性，并提供了类型安全与多平台兼容性，是构建现代 AI 应用的基础设施。

## 3. [shadcn/ui](https://github.com/shadcn-ui/ui)
> [!abstract] 一套基于 Radix UI 和 Tailwind CSS 构建的精美、可复用的 UI 组件集，旨在通过源码拷贝而非包依赖的方式提供极高的定制化能力。

### 💡 核心特性
- 基于 Radix UI 确保顶级的无障碍访问支持 (Accessibility)
- 利用 Tailwind CSS 实现极简且灵活的原子化样式控制
- 采用代码所有权模式，允许开发者直接修改组件源码以满足特定需求

### 🛠️ 技术栈
`React`, `Next.js`, `Tailwind CSS`, `Radix UI`, `TypeScript`

### ⭐ 亮点分析
它颠覆了传统 UI 组件库作为 NPM 依赖的黑盒模式，通过 CLI 自动将源码注入项目，彻底解决了第三方组件库难以深度定制和样式冲突的痛点。
